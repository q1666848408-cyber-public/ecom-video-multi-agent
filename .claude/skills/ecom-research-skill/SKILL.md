# ecom-research Skill

> Showcase version — abbreviated.

## Purpose

Wrap the research workflow into a single skill that researcher agents can invoke.

## Inputs

- `category` (str) — TikTok Shop category
- `region` (str) — country code (default: US)
- `top_n` (int) — number of products to return

## Outputs

A structured research brief in Feishu, with:
- product details
- top 5 competitor videos summarized
- script angle recommendations
