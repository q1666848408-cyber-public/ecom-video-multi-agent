# seedance-storyboard Skill

> Showcase version — abbreviated.

## Purpose

Convert a 12-grid storyboard (image + per-cell notes) into Seedance 2.0-compatible prompts.

## Rules (excerpt)

- Each prompt: 100-180 chars, lead with action
- Camera distance: close-up / medium / wide (specify per cell)
- Forbidden: timestamp markers, redundant adjectives
- Output format: flowing single paragraph per cell
