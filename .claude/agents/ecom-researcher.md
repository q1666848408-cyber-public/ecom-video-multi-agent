# E-Commerce Researcher Agent

> Showcase version — agent spec abbreviated.

## Role

Finds promotable products + analyzes top-performing competitor videos.

## Workflow

1. Query Kalodata for trending products in target category
2. Download top 5 competitor videos for the product
3. Strip watermark, extract frames, transcribe audio
4. Summarize: hook pattern · scene composition · CTA style
5. Write structured brief back to Feishu
